== Description ==

Get most affordable native app for your wordpress blog with push notifications.

<strong>Features </strong>

1. Native app for your WP Blog at most affordable price.
   For more information visit <a href="http://myandroidapp.ahsan.pk">MyAndroidApp</a> 

2. Send Push Notification for new post.

3. Monetize with Admob.

4. Send Push Notification to all users for announcement.

5. Displays Total Number of Mobile devices that have installed your app.

<strong>Demo App</strong>
Demo app is available on Google Playstore.

<a href="http://myandroidapp.ahsan.pk">Click here for more details</a> 

== Installation ==



1. Upload `myandroidapp` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enter your GCM Server Key for sending Push Notifications.
